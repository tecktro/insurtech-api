## Description

InsurTech Api made with [Nest](https://github.com/nestjs/nest) a  TypeScript framework.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash

# e2e tests
$ yarn run test:e2e

```
